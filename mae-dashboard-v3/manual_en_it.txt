MAE Dashboard V3 - Technical and User Manual (EN + IT)
Version: 1.0
Project: mae-dashboard-v3
Date: 22 Mar 2026
Reference URL: https://www.maeservice.it/newui/

1) PROJECT OVERVIEW (EN)
MAE Dashboard V3 is a web dashboard used to monitor datalogger devices in near real time.
It provides device selection, KPIs, charts, table views, GPS map support, power monitoring,
alert thresholds, export tools, and API-driven integrations.

2) PANORAMICA PROGETTO (IT)
MAE Dashboard V3 e una dashboard web per monitorare dispositivi datalogger quasi in tempo reale.
Fornisce selezione dispositivo, KPI, grafici, tabelle, supporto mappa GPS, monitoraggio
alimentazione, soglie allarmi, export e integrazioni API.

3) TECHNOLOGY STACK (EN)
Programming languages:
- HTML5
- CSS3
- JavaScript ES6+

Frameworks/libraries:
- No frontend framework (no React/Vue/Angular)
- No mandatory build framework
- Browser native APIs (Fetch API, DOM, SVG)
- Local dev server via live-server scripts

Architecture:
- state.js: global state and initialization
- api.js: API layer and normalization
- render.js: UI rendering
- modals.js: modal workflows and interactions

7) NEW APIs INCLUDED (EN)
- File list endpoint:
  /api/v1/devices/:id/files/from/:from/to/:to/type/:type/limit/:limit/offset/:offset
  Supports type filter: evt | cir | day and pagination metadata.
- File download endpoint:
  /api/v1/devices/:id/file/:name
  Supports base64 payload download.
- Event details endpoint:
  /api/v1/devices/:id/event/:evt_id
  Shows event JSON details.

UI additions:
- New "Device Files & Events" modal in Export
- Filters (from/to/type), paginated file list, per-file download
- Event details input + one-click from evt rows

Device Info additions:
- IP field = ip + port
- Public IP field = ip_public + port_public
